# CCA-F Practice Deck

Free practice questions and study tools for the **Claude Certified Architect – Foundations (CCA-F)** exam — Anthropic's official certification for architects and engineers who design and build production-grade applications with Claude, the Claude Agent SDK, Claude Code, and MCP.

This repo is a single self-contained `index.html` — no install, no account, no sign-up, no build step. Download it and open it in any browser, or use the hosted version below.

**[Open the app](https://vicsz.github.io/ccaf-exam/)**

## What's inside

- **Practice Mode** — all 400 questions, randomized, with full explanations and reveal-answer-any-time.
- **Test Mode** — a 60-question timed test that mirrors the real exam format, with a scored summary and missed-question review at the end.
- **Cheat Sheet** — a fast trigger → pattern reference for the rules of thumb and distractor patterns that come up most, grouped by domain.

Every question includes the correct answer, why each wrong choice is wrong, and transferable "when X, do Y" rules of thumb — not just answer keys.

## CCA-F exam at a glance

- **Format:** 60 scenario-based multiple-choice questions, 120-minute limit
- **Scoring:** scaled 100–1,000, passing score 720
- **Fee:** $125 USD
- **Delivery:** online-proctored or test center
- **Validity:** 12 months from award
