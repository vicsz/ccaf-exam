# CCA-F Practice Deck

A free, single-file study app for the "Claude Certified Architect – Foundations" exam — 400 practice questions with explanations, a 60-question timed test mode, and a quick-reference cheat sheet. No install, no account, no build step.

**[Open the app](https://vicsz.github.io/ccaf-exam-prep/)** — or just download `index.html` and open it in any browser.

Unofficial and community-made. Not affiliated with or endorsed by Anthropic.
